# Notification Service

A Django-based notification service that handles email and SMS notifications for e-commerce events.

## Features

- Send email and SMS notifications
- Consume events from RabbitMQ
- Store notifications in MongoDB
- REST API for manual notification sending
- Health checks and monitoring
